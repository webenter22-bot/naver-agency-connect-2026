SESSION 3

파트너의 역할과 전략 전환

발표: 최흥진 실장 (채널 컨설팅실) · "실전 노하우 공약집"

핵심 질문: "AI·에이전트가 다 하면 파트너사는?"

지금 — 사람이 운영

  • 예산 배분 · ROAS 관리 · 실시간 CPC 조정
  • 이미지 소재 생성 · A/B 테스트
  • 성과 리포트 추출·가공·인사이트
  • 야간/주말 on·off, 끝없는 소재 요청 대응
→

앞으로 — AI Agent가 자동

  • 입찰 자동 최적화(24/7 무중단)
  • AI 크리에이티브 소재·A/B 자동화
  • 이상 감지·자동 리포트·인사이트
  • Perception → Reasoning → Action 자율 운영
네이버의 선택: 광고주를 직접 보지 않고 "파트너와 함께" 만든다 — 플랫폼이 파트너를 패싱하지 않겠다는 방향.

네이버가 준비하는 4가지 지원

① 원페이 통합 계약

7월 재계약 시 MCP 포함 1브랜드=1계약. SA·DA 분리(AS-IS) → 통합 정보·데이터 → AI 통합 최적화(TO-BE)

② 강화된 API

5월: GFA 호출량·Client ID·대량 소재 등록/편집 확대 / 3분기~: ADVoost 입찰·예산·on·off·리포트

③ AEO 진단 도구

AI가 사이트 못 읽으면 광고 선출 불가. 4월 베타·무상 → 하반기 전체. 진단 + 개선 기술지원

④ AI 검수

소재 급증 대응 → 검수 대기시간 단축, 반려 사유 빠른 안내

AEO 진단 리포트: 등급(A/B/C) + 키워드 추출 여부 표시. C등급 사례 = meta description 누락 + 이미지 alt 없음 → AI가 못 읽음. 개선 후 신규 키워드·이미지까지 광고 활용.

운영 자동화 성장 4단계

STAGE 1 수동 운영
STAGE 2 부분 자동화
STAGE 3 공용 파이프라인 자동화
STAGE 4 ★ AI 기반 자율 운영(성장까지 자동화)

→ "운영은 AI에게, 사람은 전략·컨설팅으로" 이동의 근거.

속도 경고 — GPT-3.5(2022) 변호사시험 하위 50% → 1년 만에 GPT-4 상위 10% 합격. 광고 시장의 변화도 그만큼 빠를 것.

현장 질문 (특히 주목)

Q. 스마트스토어엔 AEO/메타태그 개념이 크지 않은데, SEO 올리는 방법이 새로 생기나? (KT 리더)
A. "스마트스토어는 네이버 최적화가 불필요하다. 네이버가 데이터를 더 정확히 안다. 오늘 말한 AEO는 광고주 독립몰(자사 사이트)에 해당하니 분리해서 봐라."
Q. 수수료를 광고비 %가 아니라 토큰/컨설팅비 형태로 바꿀 수 있나? (이지버즈 대표)
A. "의미 있는 의견. AI 광고 수수료를 어떻게 할지는 이제 막 고민 시작 단계. 계속 의견 듣겠다." · AEO 진단 도구는 외부 솔루션 업체와 협업(특정 업체 고정 아님, 추후 확대 예정).

파트너사가 준비할 것 (세미나 권고사항)

  1. AI 브리핑 베타 선점 — 대상 계정(ADVoost 검색/쇼핑·쇼핑검색 운영) 점검, 자사몰 랜딩 AEO 작업 (7/2 마감)
  2. AEO 진단도구 신청 — 광고주 사이트 진단 → C/B등급 개선(meta description·이미지 alt·구조화)
  3. 자동입찰 ON + 전환 트래킹 구현 — AI 학습 전제. 입찰 잦은 변경·on/off 자제
  4. 신규 광고주 계정 통합 — 7월 원페이 통합계약 대비
  5. PA 컨설팅 신청 — 미신청 파트너 다수, 초반 레퍼런스 확보
  6. Naver Ads Expert 인증 / 신규 AI 상품 웨비나 참여 (3분기 출시 대비)
  7. 역할 전환 인식 — "캠페인 운영 잘하기"에서 "광고주 사이트의 AI 최적화 진단·설득"으로
